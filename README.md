## Quán Cà phê 
### The Local Beans
- Địa chỉ: [84 Châu Thị Vĩnh Tế](https://goo.gl/maps/G6bNML96hS74Aetm6)
- Giá: 85k/ngày, free 1 ly nước bất kỳ
- Phù hợp:
